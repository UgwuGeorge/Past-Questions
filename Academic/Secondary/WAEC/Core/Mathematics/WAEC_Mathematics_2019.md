**1.** A student measures a piece of rope and found that it was 1.26m long. If the actual length of the rope was 1.25m, what was the percentage error in the measurement?
   A) 0.25%
   B) 0.01%
   C) 0.80%
   D) 0.40%
   **Answer: C**
   *Explanation: Actual length of rope = 1.25m

Measured length of rope = 1.26m

error = (1.26 - 1.25)m - 0.01m

Percentage error = error/actual length x 100%

= 0.01/1.25 x 100%

= 0.8%*

   B) 3/15
   C) 7/15
   D) 13/15
   **Answer: D**

 not live = 1/3


Wife lives = 3/5
 not live = 2/5
P(at least one lives to 80 years) = P(man lives to 80 not woman) + P(woman lives to 80 and not man) + P(both live to 80)

P=(2/3 × 2/5) + (2/5 × 1/3) + (2/3 × 3/5)

= 4/15 + 3/15 + 6/15


= 13/15*

**7.** A bricklayer charges ₦1,500 per day for himself and ₦500 per day for his assistant. If a two bedroom flat was built for ₦95,000 and the bricklayer worked 10 days more than his assistant, how much did the assistant receive?
   A) N20,000
   B) N28,000
   C) N31,200
   D) N41,000
   **Answer: A**
   *Explanation: Let the number of days worked by the assistant = t

∴
 The bricklayer worked (t + 10) days.

1500(t + 10) + 500(t) = N 95,000

1500t + 15,000 + 500t = N 95,000

2000t = N 95,000 - N 15,000

2000t = N 80,000

t = 40 days

∴
 The assistant worked for 40 days and received N (500 x 40)

= N 20,000*
