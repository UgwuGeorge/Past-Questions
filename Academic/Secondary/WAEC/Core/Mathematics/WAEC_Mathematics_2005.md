**2.** W is directly proportional to U. If W = 5 when U = 3, find U when W = 2/7
   A) 6/35
   B) 10/21
   C) 21/10
   D) 35/6
   **Answer: A**
   *Explanation: W∝U.   W=KU.  K = W/U. K = 5/3.    W = 5/3U.   2/7=5/3U.  U = 2/7 × 3/5.   U = 6/35*

**3.** Find the median of 4, 1, 4, 1, 0, 4, 4, 2 and 0
   A) 0
   B) 1
   C) 2
   D) 4
   **Answer: C**
   *Explanation: 4, 4, 1, 0, 4, 4, 2, 0
Arrange in ascending order:
0, 0, 1, 1, 2, 4, 4, 4, 4
Position of median = N+1 / 2 = 9+1 / 2 = 5th
Median = 2*
