**1.** A binary operation ⊗ defined on the set of integers is such that m⊗n = m + n + mn for all integers m and n. Find the inverse of -5 under this operation, if the identity element is 0?
   A) -5/4
   B) -5/6
   C) 0
   D) 5
   **Answer: A**
   *Explanation: m⊗n=m+n+mn
Let the inverse of -5 be n.

∴−5 ⊗ n = 0
−5 + n + (−5n) = 0
n − 5n = 5 ⟹ −4n = 5
n= − 5/4*

**3.** A car travels from calabar to Enugu, a distance of P km with an average speed of U km per hour and continues to benin, a distance of Q km, with an average speed of Wkm per hour. Find its average speed from Calabar to Benin
   A) (p+q)/(pw + qu)
   B) (uw(p+q))/(pw + qu)
   C) uw(p+q)/pw
   D) uwpw+qu
   **Answer: B**
   *Explanation: Average speed = totalDistance/TotalTime


from Calabar to Enugu in time t1, hence

t1 = P/U
Also from Enugu to Benin

t2 = q/w


Av. speed = (p + q)/(t1 + t2)


= (p + q)/((p/u) + (q/w))


= p + q x (uw)/(pw + qu)


= (uw(p+q))/(pw + qu)*
