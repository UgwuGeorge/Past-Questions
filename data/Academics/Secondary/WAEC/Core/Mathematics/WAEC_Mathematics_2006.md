**8.** If y = x sinx, find dy/dx
   A) sin x - x cosx
   B) sinx + x cosx
   C) sinx - cosx
   D) sinx + cosx
   **Answer: B**
   *Explanation: If y = x sinx, then

Let u = x and v = sinx

du/dx = 1 and dv/dx = cosx

Hence by the product rule,

dy/dx = v (du/dx) + u(dv/dx)

= (sin x) x 1 + x cosx

= sinx + x cosx*
