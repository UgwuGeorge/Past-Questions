# Data Processing (2023)

**1.** Which of the following is a DBMS software?
   A) Oracle
   B) MS Word
   C) CorelDraw
   D) Mozilla Firefox
   **Answer: A**

**2.** The process of converting raw data into meaningful information is
   A) Data collection
   B) Data processing
   C) Data storage
   D) Data transmission
   **Answer: B**

**3.** A field that uniquely identifies a record in a database is called a
   A) Foreign key
   B) Primary key
   C) Secondary key
   D) Super key
   **Answer: B**
