# Computer Studies (2023)

**1.** The physical components of a computer are referred to as
   A) Hardware
   B) Software
   C) Liveware
   D) Firmware
   **Answer: A**

**2.** Which of the following is a volatile memory?
   A) ROM
   B) RAM
   C) Hard Disk
   D) Flash Drive
   **Answer: B**

**3.** HTML stands for
   A) Hyper Text Markup Language
   B) High Tech Machine Language
   C) Hyperlinks and Text Markup Language
   D) Home Tool Markup Language
   **Answer: A**
