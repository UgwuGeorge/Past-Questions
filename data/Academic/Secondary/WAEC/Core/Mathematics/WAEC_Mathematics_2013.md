**8.** If the mean of the numbers 0, (x+2), (3x+6), and (4x+8) is 4, find their mean deviation.
   A) 0
   B) 2
   C) 3
   D) 4
   **Answer: C**
   *Explanation: Mean = {0 + (x+2) + (3x+6) + (4x+8)}/4 = 4
=> 0 + (x+2) + (3x+6) + (4x+8) = 16
8x + 16 = 16
x = 0
Now prepare a table showing the deviation of each of 0, (x+2), (3x+6) and (4x+8), adding the deviations will give 12.
Thus M.D = 12/4 = 3*
   *(Note: Closest available year from ALOC database: 2000)*

**10.** If y = x sinx, find dy/dx when x = π/2.
   A) -π/2
   B) -1
   C) 1
   D) π/2
   **Answer: C**
   *Explanation: y = x sin x
dy/dx = 1 x sinx + x cosx
= sinx + x cosx
At x = π/2, = sin (π/2) + (π/2) cos (π/2)
= 1 + (π/2) x (0) = 1*
