= x<sup>2</sup> − x − 6 ≤ 0

Now set it = 0 and factor and solve like normal.
= x<sup>2</sup> − x - 6 =0
= (x − 3)(x + 2) = 0
x + 2 = 0 or x - 3 = 0
x = -2 or x = 3...*
