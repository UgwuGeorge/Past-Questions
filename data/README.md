# Central Past Questions Database

This directory serves as the master repository for all past questions data and datasets used by the AI Exam Platform.

## Directory Structure

- `/JAMB`: Joint Admissions and Matriculation Board (Nigeria)
- `/WAEC`: West African Examinations Council
- `/NECO`: National Examinations Council (Nigeria)
- `/ICAN`: Institute of Chartered Accountants of Nigeria
- `/IELTS`: International English Language Testing System
- `/templates`: Standardized JSON/CSV templates for new data ingestion

## Standards

All questions added to this repository should follow the standardized JSON schema to ensure compatibility with the automated ingestion pipelines.
